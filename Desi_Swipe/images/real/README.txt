ECHTE BILDER
============

Lege hier alle echten Fotos ab.

Dateinamen-Tipps:
- Keine Leerzeichen (benutze _ oder -)
- Keine Umlaute (ä, ö, ü, ß)
- Beispiel: person01.jpg, gruppe_strand.jpg

Danach in index.html bei REAL_IMAGES eintragen:
  const REAL_IMAGES = [
    'person01.jpg',
    'gruppe_strand.jpg',
  ];
